# ndn-cert-gen

A command-line tool for generating self-signed certificate.

```
Usage of ./ndn-cert-gen:
  -file string
    	file name for private key and certificate (default "default")
  -identity string
    	identity (default "/ndn/guest/alice")
  -type string
    	[ rsa | ecdsa | hmac ] (default "rsa")
```
